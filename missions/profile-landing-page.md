# MISSION : Optimisation Landing Page Profil LinkedIn

## Objectif
Transformer le profil LinkedIn de l'utilisateur en une **Landing Page haute conversion** pour le produit **DeptFlow LinkedIn**.

## Instructions pour Hermes
1.  **Analyse de Contexte** : Lis attentivement `brain/deptflow-strategy.md` et `brain/branding-guidelines.md`.
2.  **Audit du Profil** : Utilise le MCP LinkedIn pour extraire le profil actuel (Headline, About, Featured, Experience).
3.  **Réécriture Stratégique** :
    -   **Headline (Titre)** : Doit exprimer la promesse DeptFlow en 1 seconde (ex: "3 à 10 prospects qualifiés dans ton CRM chaque matin. Zéro copier-coller.").
    -   **About (Infos)** : Utilise le framework "Problème - Solution - Preuve - CTA". Doit parler au profil "Freelance/Consultant B2B".
    -   **Featured (Ma sélection)** : Propose 3 types de posts à épingler (Preuve, Éducation, Histoire Fondateur).
    -   **Experience** : Réécrir l'expérience DeptFlow comme un département autonome (SDR IA).
4.  **Design de Bannière** : Décris précisément ce que doit contenir la bannière (Logo, Chiffres clés, Fond #0A0A0F, Police Outfit).
5.  **Audit Photo de Profil** : Donne des conseils pour que la photo de profil projette "Professionalisme Premium".

## Livrable
Produis un rapport complet avec les nouvelles versions de chaque section, prêtes à être copiées-collées, ou appliquées directement si l'utilisateur le demande.

## Guide de Style
-   Pas de jargon ("LinkedIn automation", "Lead generation" -> "Département de prospection autonome").
-   Style direct, impactant, premium.
-   Mise en avant du bouton "Valider / Refuser".
