# Hermes Agent Cloud
Ce repository permet de faire tourner **Hermes Agent** gratuitement via GitHub Codespaces.

## Comment démarrer ?
1. Cliquez sur le bouton **Code** (en haut à droite).
2. Allez dans l'onglet **Codespaces**.
3. Cliquez sur **Create codespace on main**.

## Une fois dans le terminal
Lancez les commandes suivantes :
```bash
source ~/.bashrc
hermes update
hermes model
```
Choisissez **Nous Portal** et suivez les instructions pour activer **MiMo-V2-Pro**.
